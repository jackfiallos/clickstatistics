Qué es ClickStatistics ?

Es una aplicación cuyo objetivo es el de analizar y mostrar la información de areas específicas sobre las cuales los usuarios son atraidos visualmente o areas en las cuales los usuarios hacen click con mayor frecuencia. La información obtenida es transformada en un mapa visual el cual permite mejorar aspectos visuales y de marketing de sitios web. Ideal para hacer estudios de mercadotecnia.

  • EyeTracking
  • HeatMap
  • Navegar por la plantilla demo y generar coordenadas
  • Resultado final con imagen de análisis
  • Esta aplicación se encuentra en desarrollo, si se desea probar mientras su desarrollo continua puedes escribir un comentario en Generacion de Eyetracking y Heatmap en Qbit Mexhico Blog.

Requerimientos

  • Javascript Frameworkd Mootools 1.2 Core y More o versiones superiores.
  • Plugin Clickstatistics.js para Mootools.
  • Servidor Apache o Windows con soporte para PHP, modulo php_gd instalado.

Instalación

  • Agregar los siguientes vinculos a los archivos entre los headers

    <script type="text/javascript" src="js/mootools-1.2-core.js"></script>
    <script type="text/javascript" src="js/mootools-1.2-more.js"></script>
    <script type="text/javascript" src="js/clickstatistics.js"></script>
    Inicializar la clase Clickstatistics

    <script type="text/javascript">
    window.addEvent('domready', function() {
    var statistics = new ClickStatistics();
    });
    </script>
